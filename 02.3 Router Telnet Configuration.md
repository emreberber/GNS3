##### 02.3 Router Telnet Configuration

```
R1# conf t
R1(config)# line vty 0 4
R1(config-line)# password 1234
R1(config-line)# login
R1(config-line)# exit
R1(config)# enable secret 5678
```

```
$ telnet 192.168.2.1
Trying 192.168.2.1...
Connected to 192.168.2.1.


User Access Verification


Password: (1234)


R1>en   
Password: (5678)
R1#
```
